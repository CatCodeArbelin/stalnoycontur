# Frontend — Стальной Контур

Next.js 15 приложение с App Router, TypeScript, Tailwind CSS, компонентами в стиле shadcn/ui и Framer Motion.

## Команды

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
